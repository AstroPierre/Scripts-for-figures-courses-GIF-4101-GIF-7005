Example of 1-D bayesian classification.¶

Let us assume that we have some random data spread in a bimodal histogram and that we would like to separate the data in two classes. How should we proceed?

For simplicity, we also assume the overall probability density function (PDF) to be a mixture of two gaussian PDFs. Thus we have:

$P(x|C_{0}) = \frac{1}{\sqrt{2\pi \sigma_{0}}} e^{-(x-\mu_{0})^2/2\sigma_{0}^2}$ with a class probability of $P(C_{0})$
$P(x|C_{1}) = \frac{1}{\sqrt{2\pi \sigma_{1}}} e^{-(x-\mu_{1})^2/2\sigma_{1}^2}$ with a class probability of $P(C_{1})$

where $P(x|C_{i})$ is the probability density of observing x in class i. The overall PDF is given by:

$P(x) = P(C_{0}) P(x|C_{0}) + P(C_{1}) P(x|C_{1})$

We can use Bayes' method to obtain the a posteriori class probability $P(C_{i}|x)$ which is the probability of class i given the observation x.

Why should we bother? Actually, we are often more interested in predicting the most probable class given an observation x than knowing the probability of observing x in a given class.

Using this information, we will be able to locate the boundaries delimiting the influence zone of each class, i.e. where $P(C_{1}|x) > P(C_{0}|x)$ or $P(C_{0}|x) > P(C_{1}|x)$

def generate_PDF():
    x = np.linspace(-10., 10., 100)

    pdf_a = scipy.stats.norm.pdf(x,mu[0],sigma[0])
    pdf_b = scipy.stats.norm.pdf(x,mu[1],sigma[1])

    return (x, pdf_a, pdf_b)

Using the density probability of observing x in each class $P(x|C_{i})$ and the a priori class probabilities $P(C_{i})$, we can compute the a posteriori probabilities $P(C_{i}|x)$. Those are the probabilities of each class given the observations x.

According to Bayes' theorem, the a posteriori probabilities are:

$P(C_{i}|x) = P(x|C_{i}) P(C_{i}) / P(x)$

with the normalization factor $P(x)$:

$P(x) = \sum_{i=1}^{2}{P(x|C_{i}) P(C_{i})}$

def compute_a_posteriori_probabilities():

    # We use the discriminant function notation h(x).The denominator is the normalization factor
    h_a = pdf_a*pC[0]/(pdf_a*pC[0] + pdf_b*pC[1])
    
    h_b = 1- h_a

    return (h_a, h_b)

Example I: PDFs with identical standard deviations and a priori class probabilities¶

This is a simple example where the individual PDFs are similar except for their mean values. Looking at the first panel below, a threshold value of $x=0$ can delineate the influence zone of each class :

Class 1 wins when $P(C_{1}|x) > P(C_{0}|x)$ i.e. when $x \geq 0$

There is no need to look at the second panel; the curves' symmetry in the first panel is enough. No need for Bayes' theorem here.

Example II: PDFs with different standard deviations and a priori class probabilities¶

This a more difficult problem. In the first panel below, both curves cross each other twice along the x axis. Hence, there are 2 thresholds delimiting the local winning class but it is not obvious what their values are.

It is a situation where Bayes' theorem becomes handy. The second panel shows very well where to put the thresholds:

Class 0 wins when the a posteriori probabilities $P(C_{0}|x) > P(C_{1}|x)$ i.e. when $x \in \left[ -9, 2 \right]$
