Examples of 2-D probability density functions estimated using histograms and KDE methods¶

In this script, we compare estimates of probability density functions (PDF) based on two popular methods:

  • Standard histogram-based approach
  • Kernel Density Estimation (KDE) with gaussian and Epanechikov kernels

The methods are applied to a dataset generated from a bivariate normal.

Standard histogram-based PDF estimator¶

In this method, the probability density $\hat P(\bf{x})$ is proportional to the number of data points $\bf{x^{t}}$ inside the joint intervals $[x_{1}, x_{1}+h[$ and $[x_{2}, x_{2}+h[$

$\hat P(\mathbf{x}) = \large{\frac{ \# \{\{x_{1}\le x_{1}^{t}< x_{1}+h \} \cap \{ x_{2}\le x_{2}^{t}< x_{2}+h \}\}}{Nh^{2}}} $

KDE PDF estimator¶

In this method, the probability density $\hat P(\mathbf{x})$ is modeled as the convolution of the data distribution with a kernel $K(u)$ designed to smooth the experimental PDF

$\hat P(\mathbf{x}) = \frac{1}{Nh}\sum_{t=1}^{N} K(\frac{x-x^{t}}{h})$

The level of smoothing is controled by the bandwidth parameter h.

Gaussian and Epanechikov are popular kernels. Their 1-D and 2-D expressions are:

1-D Gaussian: $K(u)= \frac{1}{\sqrt{2\pi}} \exp [-\frac{u^{2}}{2}]$

2-D Gaussian: $K(\bf{u})= \frac{1}{2\pi} \exp [-\frac{\|u\|^{2}}{2}]$


1-D Epanechikov: $K(u)= \frac{3}{4}(1-u^{2})$ for $u \in [-1, 1]$

2-D Epanechikov: $K(\bf{u})= \frac{15}{8\pi}(1-\|u\|^{2})$ for $u \in [0, 1]$

In this script, we only use the 2-D versions of the kernels.

def compute_KDE_pdf(npts, h_gaussian, h_epanechikov, X, xx, Xgrid):

    pdf_g = np.zeros(xx.shape)
    pdf_e = np.zeros(xx.shape)

    for n in range(npts):    
        dX = (X[n,:] - Xgrid)
        dX2 = dX[:,0]**2 + dX[:,1]**2
        
        u2 = dX2/h_gaussian**2
        pdf_g = pdf_g + gaussian_2D(u2).reshape(xx.shape)

        u2 = dX2/h_epanechikov**2
        pdf_e = pdf_e + epanechikov_2D(u2).reshape(xx.shape)
        
    # Normalisation
    pdf_g = pdf_g/(npts*h_gaussian**2)
    pdf_e = pdf_e/(npts*h_epanechikov**2)
    
    return (pdf_g, pdf_e)
        

# KDE kernels
def gaussian_2D(u2):
    y = np.exp(-u2/2)/(2*np.pi)
    return y

def epanechikov_2D(u2): 
    y = (15/(8*np.pi))*(1-u2)
    indx = np.where(u2>=1)
    y[indx] = 0
    return y

def compare_conditional_PDF(figName):
    fig, axs = plt.subplots(1,2,figsize=(15,5))

    axs[0].imshow(pdf_true, interpolation='nearest', origin='low',
        extent=[xedges[0], xedges[-1], yedges[0], yedges[-1]], cmap=cm)
    axs[0].plot(xx[0,:],2*np.ones((nbins,),dtype=int), ls='-', color='g', lw=3)
    axs[0].set_title('True PDF', fontsize=18, color='k')
    axs[0].set_xlabel('$x_{1}$', fontsize=14)
    axs[0].set_ylabel('$x_{2}$', fontsize=14, rotation=0)
    axs[0].yaxis.set_label_coords(-0.15, 0.5) 
    axs[0].grid(None)  

    i = int(nbins/2)
    u = xx[0,:]
    
    axs[1].plot(u, pdf_true[i,:] , ls='-', color='k', label='True')
    axs[1].plot(u, pdf_hist[i,:] , ls='-', color='r', label='Histogram')
    axs[1].plot(u, pdf_KDE_gauss[i,:] , ls='-', color='g', label='KDE gaussian')
    axs[1].plot(u, pdf_KDE_epa[i,:] , ls='-', color='b', label='KDE Epanechikov')
    axs[1].set_xlim([1., 3.])
    axs[1].set_xlabel('$x_{1}$', fontsize=14)
    axs[1].set_ylabel('Probability density', fontsize=14)
    axs[1].set_title('$P(x_{1}|x_{2}=2)$', fontsize=18, color='k')

    axs[1].legend(loc='upper right')

    fig.tight_layout()
    plt.savefig(figName + '.png')
    plt.savefig(figName + '.pdf')
